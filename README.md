# financial-literacy-regression-analysis
Applied econometric analysis of financial literacy using a multiple linear regression model (OLS). The project examines the impact of age, gender, education, and number of children, using dummy variables for categorical data. Includes model estimation, statistical testing, and interpretation of results using R.
## Key Results
- Age has a positive and significant effect
- Education level strongly impacts financial literacy
- Gender differences are statistically significant
- Model is statistically valid (F-test p < 0.001)
